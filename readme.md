# React Boilerplate 

В данном бойлерплейте установлен и настроен webpack для продакшн и дев окружения.

## Используемые пакеты

- React 17.0.2
- Typescript 4.9.4
- StyledComponents 5.3.6(несовместим с react@18 и выше)
- Webpack 4
- Babel 7

## Запуск dev среды 

npm start

## Билд

npm run build

## Конфигурация

- webpack.dev.config.js - dev среда
- webpack.prod.config.js - prod среда
